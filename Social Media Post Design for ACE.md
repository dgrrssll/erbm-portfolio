# Social Media Post Design for ACE

**Client:** ACE
**Project Type:** Social Media Post Design
**Role:** Graphic Designer

Created a one-time social media promotional post for ACE to support the ACE x Sweathauz event. The design was made to help promote the collaboration, highlight the event clearly, and create an engaging visual for online audiences.

## Key Work

* Designed a social media post for the ACE x Sweathauz event
* Created a clean and engaging promotional layout
* Highlighted the event details in a clear and readable format
* Used visual elements aligned with the event and brand presentation
* Prepared the design for online posting and social media promotion
