# Fetch & Boxed Website UI/UX

A website UI/UX design concept for **Fetch & Boxed**, a pet subscription website for dog owners. This project was designed using **Canva**.

## About the Project

Fetch & Boxed is a pet subscription website that allows customers to either customize their own subscription box or buy a curated box for their dogs.

The goal of this design is to create a simple, friendly, and easy-to-use shopping experience for pet owners who want convenient access to dog treats, toys, and essentials.

## Features

* Custom dog subscription box option
* Curated dog box option
* Clean and pet-friendly website layout
* Simple navigation
* Clear call-to-action buttons
* Product-focused sections for dog treats, toys, and essentials
* Beginner-friendly e-commerce flow

## Target Users

This design is intended for:

* Dog owners
* Pet lovers
* Busy customers who want convenient pet shopping
* Customers who prefer subscription-based pet products

## Tools Used

* Canva

## Project Type

* Website UI/UX Design
* Pet Subscription Website
* E-commerce Design Concept

## Design Goal

The main goal of Fetch & Boxed is to make dog subscription shopping simple and enjoyable. Customers can choose between building their own box or selecting a ready-made curated box based on their dog’s needs.
