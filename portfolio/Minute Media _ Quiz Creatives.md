# Quiz Creative Design for Minute Media

**Client:** Minute Media
**Project Type:** Quiz Creative Design
**Role:** Graphic Designer

Created quiz creatives for Minute Media to support interactive and engagement-focused online content. The graphics were designed to attract attention, encourage clicks, and make quiz topics visually interesting for viewers.

## Key Work

* Created graphics for quiz-based online content
* Designed attention-grabbing visuals to encourage user interaction
* Used clear layouts and readable text for quick topic recognition
* Created engaging designs for entertainment, lifestyle, trivia, and knowledge-based quiz topics
* Prepared creatives for digital publishing and audience engagement
