# Website UI/UX Design for Fetch & Boxed

**Client:** Fetch & Boxed
**Project Type:** Website UI/UX Design
**Role:** UI/UX Designer/Graphic Designer

Designed a website UI/UX concept for Fetch & Boxed, a pet subscription box brand for dog owners. The website mockup was created to present a simple, friendly, and customizable shopping experience where users can choose subscription boxes based on their dog’s size, style preference, and needs.

## Key Work

* Created a website mockup for a pet subscription box brand
* Designed a clean, playful, and pet-friendly layout
* Organized subscription box options such as Basic, Deluxe, Premium, and Birthday Box
* Used clear call-to-action sections and simple e-commerce navigation
* Designed the concept using Canva for online presentation
