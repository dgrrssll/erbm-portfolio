# Search Creative Design for Minute Media

**Client:** Minute Media
**Project Type:** Search Creative Design
**Role:** Graphic Designer

Created search creatives for Minute Media to support website content across different high-search topics. The graphics were designed to attract clicks, communicate the topic clearly, and help users quickly understand the content being promoted.

## Key Work

* Created graphics for search-based website content
* Designed creatives for topics such as personal loans, truck driving jobs for seniors, and asbestos removal from homes
* Used clear and direct visual layouts for quick topic recognition
* Created attention-grabbing designs for informational and lead-focused content
* Prepared graphics for online publishing and search-driven audience engagement
